DNS Manager - Hilfedokumentation

Funktionen

1. Manager

Ändere Deine Dns auf welches Server den du willst

  • Jeder Server ist anderst und hat auch immer eine andere main Funktion
  • z.b. ist einer für Geschwindigkeit und einer für Sicherheit

2. Verfügbare Server

  • Google (Optimiert für Geschwindigkeit)
  • Cloudfare (Optimiert für Privatsphäre)
  • Quad9 (Optimiert für Sicherheit)
  • OpenDNS (Optimiert für Filterung)
  • AdGuard (Optimiert für Werbeblockierung)
  • Alternate DNS (Optimiert für Geschwindigkeit)
  • CleanBrowsing (Optimiert für Familienfreundlichkeit)

Schnellstart-Anleitung

  1. Zip Datei Herunterladen
  2. Zip Datei extrahieren
  3. Ordner öffnen
  4. dns-manager.exe ausführen
  5. Gewünschte DNS auswählen und benutzen
